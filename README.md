# steezy
Steezy is a python tool for ghetto generation of Yara rules from the VA
of a function in a compiled executable.

## Installation
* Install dependancies.
* Clone git repo.
* Hope it works.

### Dependancies
Python 3.x (Tested with 3.7+).
